Новости

Возможности

— Получение звука, приближённого по качеству к оригинальному методом воспроизведения через виртуальный аудиовыход, предназначенный для аудио высокого разрешения без лишних обработок, включенных в системе по умолчанию

Реализация

Для внесение изменений в систему Android требуется ROOT-доступ в виде установленного Magisk.

Внесение изменений производится установкой аудиомодифкации в Magisk Manager, либо ручной правкой системных файлов проводником с поддержкой ROOT-доступа.

Поддерживаемые устройства

Аудмодификации и отдельные гайды для улучшения качества звука на Android-устрйствах подходят практически под все смартфоны на базе Qualcomm и MediaTek процессоров.

В случае неработоспособности какой-либо из аудиомодификаций на вашем смартфоне можно воспользоваться гайдами и вручную внести изменения в системные файлы.
